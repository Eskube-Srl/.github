<h3 align="center">Systems Safety & Security</h3>

---

Eskube builds AI-assisted tools and systems for safety-critical and security-sensitive environments.

All projects follow the **[Eskube AI Development Standard](https://github.com/Eskube-Srl/eskube-project-template/blob/main/eskube/claude-code-guidelines.md)** — a shared ruleset covering architecture, code quality, testing, and auditability.

## Starting a new project

→ **[Create from template](https://github.com/new?template_name=eskube-project-template&template_owner=Eskube-Srl)**

**Developers:** follow the [Getting Started guide](https://github.com/Eskube-Srl/eskube-project-template/blob/main/eskube/getting-started.md)  
**Non-developers:** open the project folder and double-click `start.bat` — the assistant will guide you from there
